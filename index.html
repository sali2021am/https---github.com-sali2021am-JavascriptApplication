Implémentation d'une solution de gestion d'identité de confiance en utilisant les mécanismes OAuth 2.0 Vs OpenID Connect Vs SAML

Dans le cadre de stage , on a choisie Keycloak comme solution Open Source de gestion des identités et des accés pour les applications et services modernes.

Keycloak est un logiciel à code source ouvert permettant d'instaurer une méthode d'authentification unique à travers la gestion par identité et par accés. Initialement développé par les équipes JBoss, le projet est depuis mars 2018 sous la gérance de Red Hat qui l'utilise en amont de sa solution RH-SSO2. Keycloak se définit comme une application rendant possible la sécurisation de n'importe quelle application web moderne avec un apport minimum en termes de code.
